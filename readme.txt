The code named “UST-vs-VDZ-code” was performed to compare the clinical and objective effectiveness between UST and VDZ at week 26 and 52. 

Code 1 named ‘’UST-vs-VDZ-in-iptw’’ was performed to compare the effectiveness between UST and VDZ at week 26 and 52 using inverse probability of treatment weighting.

Code 2 named “UST-vs-VDZ-in-psm” was performed to compare the effectiveness between UST and VDZ at week 26 and 52 using propensity score matching as sensitivity analysis.
